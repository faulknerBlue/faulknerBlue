SoniCelegans and Beyond

Multimedia Installation and Performance Exhibit - 09.26.2021

Soni C. Elegans explored the machinations of memory through the lens of Dr. Eleni Gourgou's research on C. Elegans nematodes. The project began entirely open-ended, giving the student team complete artistic license and direction. Ultimately, we created an immersive exhibit which featured a multimedia performance and a pre-performance installation. We built motion-activated scent-jars and a wall of tactile textures which, when approached by audience members, triggered events in a generative music piece in Max MSP for the preperformance instllation. The main performance featured a generative Graphic Score which utilized data directly from Dr. Gourgou's research in concert with live Max MSP analyses of our performance. The backdrop for the score was AI generated video and images, which used text inputs from the audience.



What I did:

- Creative Programmer: coded a generative Graphic Score/visual which employed AI generated images in Processing, co-developed Max MSP live audio processing patch which informed the visuals

- Audio Engineer: designed the signal flow for the performance component with live effects processing and ambisonic playback

- Visual Artist: created a looping visual for the performance installation

- Sound Artist: live performance on Modular Synthesizer
