SoniCelegans and Beyond

Multimedia Installation and Performance Exhibit - 09.26.2021

Soni C. Elegans explored the machinations of memory through the lens of Dr. Eleni Gourgou's research on C. Elegans nematodes. Initially, the prompt for the project was entirely open, so it was up to us on the 4-student team to design an exhibit from the ground up. Ultimately, we created an immersive exhibit with both a multimedia performance and a pre-performance installation. The installation included motion-activated scent-jars and a wall of tactile textures which, when approached by audience members, triggered events in a generative music piece in Max MSP.



What I did:

- Creative Programmer: coded a generative Graphic Score/visual which employed AI generated images in Processing, co-developed Max MSP live audio processing patch which informed the visuals

- Audio Engineer: designed the signal flow for the performance component with live effects processing and ambisonic playback

- Visual Artist: created a looping visual for the performance installation

- Sound Artist: live performance on Modular Synthesizer
